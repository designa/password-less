Registrando sites

Visão global

O diagrama de sequência na direita mostra o esquema geral de autenticação. Ele contém os seguintes passos:

  • Depois de registrar o seu site, você pode fazer uma chamada HTTP POST ao Password-Less com e-mail de usuário ou id;
  • Password-Less envia um e-mail para o usuário com um link de login seguro;
  • O usuário clica no link de acesso;
  • Password-Less envia uma chamada HTTP GET para o seu site para que então você saiba que logou;
  • Seu site envia uma chamada HTTP GET para o Password-Less perguntando pelo ID de usuário e e-mail;
  • Após receber dados do usuário, você loga seu usuário e aparece sua página de bem-vindo;

Cada passo, incluindo parâmetros e formatos de resposta, serão explicados com detalhes nas próximas seções.

O sumário mostra todas as REST api's disponíveis. O formato retornado é JSON.
